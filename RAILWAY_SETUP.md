# Настройка PostgreSQL на Railway

## Шаги:

1. Зайди в дашборд Railway: https://railway.app
2. Открой свой проект `sport_bot`
3. Нажми "New" → "Database" → "Add PostgreSQL"
4. Railway автоматически создаст переменную окружения `DATABASE_URL`
5. Бот автоматически подключится к PostgreSQL при следующем деплое

## Локальная разработка:

Для локального тестирования используется SQLite (уже настроено в .env):
```
DATABASE_URL=sqlite+aiosqlite:///./sport_bot.db
```

## Что изменилось:

- ✅ Все игры теперь хранятся в БД (таблица `games`)
- ✅ Завершённые игры в БД (таблица `completed_games`)
- ✅ Пользователи в БД (таблица `users`)
- ✅ Данные не пропадают при перезапуске
- ✅ Готово к работе с 200+ пользователями

## Миграция данных:

Старые игры из `games_storage` (список в памяти) не мигрируются автоматически.
При первом запуске с БД бот начнёт с чистого листа.
